# ADR 0001: Record Architecture Decisions

**Status:** Accepted  
**Date:** 2026-07-06

## Context

SentinelANPR AI requires traceable decisions as the system evolves (ML models, databases, APIs).

## Decision

We will use Architecture Decision Records in `docs/adr/`, one file per decision, numbered sequentially.

## Consequences

- **Positive:** Historical context preserved; onboarding easier
- **Negative:** Small documentation overhead per decision
